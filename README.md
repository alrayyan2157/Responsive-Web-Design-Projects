# Responsive Web Design Projects

This repository contains a collection of projects completed as part of the [freeCodeCamp Responsive Web Design Certification](https://www.freecodecamp.org/learn/).  
Each project demonstrates core web development skills using HTML and CSS.

   [View Project](./Batman-tribute-page/) | [Live Demo](https://codepen.io/alrayyan2157/full/raOVzww)

   [View Project](./solar-survey-form/) | [Live Demo](https://codepen.io/alrayyan2157/full/WbQvEzw)

5. **Personal Portfolio Page**  
   A portfolio showcasing all the above projects and providing contact information.  
   [View Project](./Personal-Portfolio-Page/) | [Live Demo](https://codepen.io/alrayyan2157/full/WbQdWrZ)
